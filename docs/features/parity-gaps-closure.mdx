# Pentest-Copilot Parity Closure

This release closes major parity gaps:

- Burp + Caido coexistence with run-level Burp panel and automation tools.
- Capability pre-installs via registry-driven installer hooks.
- MCP server/client foundations (gallery, custom servers, PAT issuance, SSE/stdio server).
- Racer-lane nova helpers for parallel subagent execution strategies.
- VPN profile upload/status endpoints for tunnel-aware sandbox workflows.
- Slash-command command palette (`/pause`, `/resume`, `/kill`, `/budget`).
- Engagement-state tools for first-run and run-lifecycle memory.
